# DS Playbook

Panduan coding dan standar kerja tim Data Science — tersedia sebagai single-file HTML.

## Cara pakai

Buka `index.html` langsung di browser — tidak perlu server atau instalasi apapun.

## Deploy ke GitHub Pages

1. Push ke GitHub
2. Aktifkan GitHub Pages di **Settings → Pages → Source: main branch, / (root)**
3. Playbook akan tersedia di `https://[username].github.io/[repo-name]/`
